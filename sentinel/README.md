# Sentinel

A prototype system for aerospace crash dump analysis and evaluation.

Structure:

- `.env.example` — example environment variables, including `OPENAI_API_KEY`
- `backend/` — FastAPI backend with LangGraph agent, RAG pipeline, simulator, and evaluation harness
- `frontend/` — React application for user interaction
- `notebooks/` — Kaggle fine-tuning notebooks and experiments
- `demo_cache/` — pre-computed demo responses
- `evaluation/` — evaluation results and JSON outputs
- `docs/` — architecture diagrams and documentation
